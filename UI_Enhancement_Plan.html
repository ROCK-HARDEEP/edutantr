FRONT-END ONLY

Branding & Animation Plan

A focused plan to make the Edutantr public website visually attractive, on-brand as a tech training company, and animated with modern effects.

June 15, 2026  |  Public Website Only  |  No Admin Panel / No Backend

Scope of This Document

This document covers only the public-facing website — what visitors, students, and companies see. It lists exactly:

  • Where the current design has branding gaps
  • What visual/style changes to make on each page section
  • Where and how to add animations to make the website attractive and catchy

Not covered: Admin panel, backend APIs, database, server configuration, or student dashboard internals.

Part 1 — Branding & Content Fixes

These are the areas where the website's look, text, and visual identity don't match a tech training company.

CRITICAL

1.1 — Hero Section (Homepage Top Banner)

Problem: Headline is vague — "From beginner to skilled — starting in minutes". Doesn't say tech, IT, or jobs.

Fix:

  • Headline → "Launch Your IT Career — Get Trained. Get Skilled. Get Placed."
  • Add tech areas below: "Full-Stack · Data Science · Cloud · Cybersecurity"
  • Add stat: "2,000+ students placed at top companies"
  • CTA button → "Start Your Tech Career"
  • Add batch urgency: "Next batch starts July 1"
CRITICAL

1.2 — Placements Section (Currently HIDDEN)

Problem: A beautiful section showing placed students exists but is turned off in code.

Fix: Enable it. Display prominently on homepage after courses. Show student photo, name, company, role, and "Placed ✓" badge.

CRITICAL

1.3 — CTA Banner (Green Section Near Bottom)

Problem: Says "Share Your Knowledge, Inspire Learners" — targets teachers, not students.

Fix: Headline → "Ready to Start Your IT Career?" / Buttons → "Enroll Now" + "Talk to a Counselor"

HIGH

1.4 — Course Cards (Missing Tech Identity)

Problem: Cards show title + price + rating only. No tech stack, no duration, no difficulty level.

Fix: Add colored tech badges (Python, React, AWS), duration label (12 Weeks), difficulty tag (Beginner/Advanced).

HIGH

1.5 — "How It Works" Steps (Generic Text)

Problem: Steps say Join → Learn → Build → Certify. Same as any course platform.

Fix: Step 1: Choose Your Tech Track → Step 2: Learn Through Live Projects → Step 3: Build Your Portfolio → Step 4: Get Placed

HIGH

1.6 — Career Tracks Section (Does Not Exist)

Problem: No structured program view. Courses shown as flat marketplace list.

Fix: Add new "Career Tracks" section on homepage. Large horizontal cards with track name, duration, course count, salary range. Example: "Full-Stack Web Developer — 16 Weeks — 8 Courses — ₹4-8 LPA"

HIGH

1.7 — Course Detail Page (Missing Info)

Fix: Add "Technologies You'll Learn" with logos, "Download Syllabus" button, "Upcoming Batches" with dates, instructor's industry background.

MEDIUM

1.8 — Navigation Menu

Fix: Add "Programs" link between Home and Courses. On hover, show dropdown with 3-4 featured career tracks.

MEDIUM

1.9 — Mobile View Polish

Fix: Larger hero text on phones, simplified course cards on small screens, collapsible footer sections.
